# Word Analyzer App (Python)

A beginner-friendly console-based application that analyzes sentences.  
This project demonstrates variables, strings, string functions, conditionals, lists, and tuples.

## 🚀 Features
- Count words and characters
- Count vowels and consonants
- Check if the sentence is a palindrome
- Store results in a tuple

## 📚 Concepts Used
- Variables & Data Types
- Strings & String Functions
- Conditional Statements
- Lists & Tuples

## ▶️ How to Run
Clone the repository and run:
```bash
python word_analyzer.py
